# Tech Symposium 2025 🎉

Static event information page for Annual Tech Symposium 2025. Built with HTML5 and CSS3.

### ✨ Features
- **Event Details** - Info about the 2-day tech symposium
- **Photo Gallery** - Highlights from past events
- **Contact Section** - Address, phone, and email for inquiries
- **Responsive Design** - Works on desktop and mobile
- **Custom Favicon** - Date icon for the tab

### 🛠️ Tech Stack
- HTML5
- CSS3 (inline styling)
- Favicon + images
